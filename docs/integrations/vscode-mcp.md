# VS Code MCP Integration

1. Install an MCP-compatible VS Code extension (e.g., Claude for VS Code).
2. Create `.vscode/mcp.json` with an entry pointing to `python -m mcp_hub.server`.
3. Provide environment variables for configs.
4. Reload VS Code and enable the server in the extension UI.
5. Use Copilot/Chat panels to call DevOps tools.
